Extra tools & support for [Bajo](https://github.com/ardhi/bajo)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install bajo-extra
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```bajo-extra``` in it
